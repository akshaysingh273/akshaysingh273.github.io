Decoding MEG signals using Deep learning

Motor and Cognitive imagery task based MEG signals for 17 subjects were processed using MNE-Python. Binary classification datasets were created using 4 classes of tasks (Hands imagery, Feet imagery, Subtraction, Word generation). I made a CNN + LSTM based deep learning model with multi-head attention, which I used for binary class classification task and for multi-class classification task as well.

Autonomous robot car

Made a Robot car equipped with Arduino Uno, ESP32 micro-controller and ultrasonic sensors. Ultrasonic sensors were used for obstacle detection and AO* path finding algorithm was used for finding a path between defined start and destination positions while actively detecting and avoiding obstacles.

Hand Gesture Controlled Robot Car

Made a hand gesture controlled Robot car using Arduino Uno and HC-05 Bluetooth module. For gesture recognition, made a gesture sensing hand glove by integrating 3-axis accelerometer, Arduino Uno and HC- 05 Bluetooth module for Bluetooth communication between the Robot car and the hand glove. The glove would recognize hand movement then convert them into control commands that were then sent to the Robot car.

Room control system through smartphone

Robotic Arm Simulator for 2R planar manipulator

Eye Tracking Software
